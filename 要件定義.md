# gmail-llm-gcal
機能要件

# UI
gmail に存在する`予定を作成`ボタンを流用する．

# ロジック
## 本来の動作
1. メールを開く
2. `予定を作成`ボタンを押す
3. 次のリンクで別タブが開く．リンクは例． https://calendar.google.com/calendar/event?action=TEMPLATE&authuser=0&dates=20260307T110000Z%2F20260307T120000Z&text=test&details=鈴木様%0A%0Aお世話になっております．%0A田中です．%0A%0A次回のミーティングは3%2F10の12%3A00-13%3A00でお願いします．%0A田中
4. 次のリンクにリダイレクトされ，予定作成画面に遷移する． https://calendar.google.com/calendar/u/0/r/eventedit?dates=20260307T110000Z/20260307T120000Z&text=test&details=鈴木様%0A%0Aお世話になっております．%0A田中です．%0A%0A次回のミーティングは3/10の12:00-13:00でお願いします．%0A田中

## 所望の動作
1. メールを開く
2. `予定を作成`ボタンを押す
3. リンクが別タブで開く前にキャッチし，独自処理を行う
4. 独自処理後のリンクが開く．
5. リンクがリダイレクトされ，予定作成画面に遷移する
